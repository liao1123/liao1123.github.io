Markdown基本语法

发表于2024-05-04|更新于2024-05-04
|字数总计:610|阅读时长:1分钟|阅读量:

注意所有的markdown语法均为英文符号

标题语法

前面多少个#号代表几级标题，标题与#号之间需要有空格，否则就不是达到效果

代码：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题

段落语法

使用空白一行达到换行的效果，其实这样子是重新开启一个段落，在行间距上会体现差异

直接换行是达不到效果的
也可以直接在上一行末尾添加两个空格然后回车就可以换行

强调语法

一个星星达到斜体效果
两个星星达到加粗效果
三个星星达到加粗和斜体的效果

代码：

*一个星星*
**两个星星**
***三个星星***

效果：

一个星星
两个星星
三个星星

引用语法

引用直接在内容前面加一个>大于号即可，也可以嵌套使用>>

代码：

> 这是一个引用
> 这也是一个引用
> > 这是一个嵌套引用

效果：

这是一个引用
这也是一个引用

这是一个嵌套引用

列表语法

1、有序列表，标号后面加上英文原点然后空格即可

代码：

1. 这是第一个
2. 这是第二个
3. 这是第三个

效果：

  1. 这是第一个
  2. 这是第二个
  3. 这是第三个

代码块语法

句中的代码用单反引号（注意区分反引号和引号区别）即可，如 code

代码：

`code`

效果：

code
